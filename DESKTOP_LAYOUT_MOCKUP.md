## Key Changes from Current Design:

### 1. **Two-Column Layout for Wind & Tides** (Priority Info)
   - Wind chart on left half
   - Tide chart on right half
   - Both visible simultaneously without scrolling
   - Each gets ~800px width on 1920px screen

### 2. **Reduced Chart Heights**
   - Wind & Tide charts: 250px â†’ 180px (more compact)
   - Temperature chart: 250px â†’ 150px (single row below)
   - Still readable but space-efficient

### 3. **Compact Day Selector**
   - Horizontal, smaller buttons (60px wide instead of 90px)
   - Less vertical space (60px total vs 100px)

### 4. **Simplified Headers**
   - Combine current conditions with chart titles
   - Remove extra padding and borders
   - More information density

### 5. **Temperature De-emphasized**
   - Moves to bottom in full-width layout
   - Smaller chart since it's less critical for fishing
   - Still visible but doesn't dominate
