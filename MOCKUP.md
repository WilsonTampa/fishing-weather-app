## Design Details

### Color Scheme (Dark Theme)
- **Background**: `#0D1117` (dark navy/black)
- **Surface/Cards**: `#161B22` (slightly lighter)
- **Text**: `#FFFFFF` (white) and `#8B949E` (light gray for labels)
- **Wind line**: `#58A6FF` (light blue)
- **Temperature line**: `#F97316` (orange) transitioning to `#3B82F6` (blue) for cooler temps
- **Tide line**: `#4A90E2` (medium blue)
- **Accent/Interactive**: `#58A6FF` (bright blue)
- **Arrows/Icons**: `#FFFFFF` with slight transparency

### Typography
- **Headers**: 18-24px, Bold, Sans-serif (Inter, SF Pro, or similar)
- **Current values**: 24-32px, Semi-bold
- **Chart labels**: 12-14px, Regular
- **Body text**: 14-16px, Regular

### Interactive Elements
- **Charts**: Hovering/tapping any point shows tooltip with exact values
- **Day Selector**: Swipe left/right on mobile, click on desktop
- **Toggle buttons**: Wind Speed/Gusts switches between views
- **Location pin**: Returns to map view
- **Tide markers**: High/low points are clearly marked on curve

### Responsive Behavior
- **< 768px (Mobile)**: Stacked vertical layout, full-width charts
- **768-1024px (Tablet)**: Similar to mobile but with more chart detail
- **> 1024px (Desktop)**: 2-column grid for wind/temp, full-width tide below

### Key Features
1. **Synchronized Time Axis**: All charts use same hourly intervals
2. **Current Indicators**: Each section highlights current conditions
3. **Clean Visual Hierarchy**: Wind â†’ Temperature â†’ Tides
4. **Minimal Scrolling**: Desktop view shows all data above the fold
5. **Touch-Friendly**: Large tap targets, swipe gestures
6. **High Contrast**: Excellent outdoor visibility

### Chart Specifications

**Wind Chart:**
- Line chart with direction arrows overlaid
- Arrows point in wind direction (e.g., â†— = from SW, going NE)
- Data points every hour
- Toggle to show gusts as dashed line

**Temperature Chart:**
- Smooth line/area chart
- Gradient fill optional (warm â†’ cool colors)
- High/low markers for the day
- Tooltip shows exact temperature on hover/tap

**Tide Chart:**
- Smooth bezier curve
- Current tide marked with circle (âŠ™)
- Covers full 24-hour period
- Tooltip shows exact tide height on hover/tap

### Interactions
- **Hover (Desktop)**: Shows tooltip with time and exact value
- **Tap (Mobile)**: Shows tooltip, persists until another tap
- **Swipe**: Changes day (left = next day, right = previous)
- **Pull-to-refresh**: Updates all data
- **Click location**: Returns to map view
